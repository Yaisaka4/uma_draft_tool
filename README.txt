# 🏁 F1 Umamusume Banpick

Ứng dụng web quản lý ban pick Umamusume với phong cách F1.

## ✨ Tính năng

- **Admin**: Quản lý users, tạo lobby, phân hạn
- **Referee**: Theo dõi 4 lobby, phân seed, bắt đầu ban pick
- **Trainer**: Đăng nhập, upload ảnh, tham gia ban pick

## 🚀 Công nghệ sử dụng

- Flask
- SQLite
- Flask-SocketIO
- HTML/CSS/JavaScript

## 📦 Cài đặt

1. Clone repository
2. `pip install -r requirements.txt`
3. `python app.py`
